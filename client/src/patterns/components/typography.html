Typography.

All typographic elements adhere to a standard system which utilises Vertical Rhythm and a set of utilites to add and remove margin to stay withing the baseline. The following rules apply.

  • 18px baseline
  • All have default bottom margins or 1 baseline unit. I.e. they default to 18px bottom margin.
